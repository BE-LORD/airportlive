## 9.1 The Three-Wow Rule

Only three moments should feel “wow”:

1. Hero title reveal
2. Route line drawing
3. Fleet/Journey cinematic scroll

Everything else should be restrained.

## 9.2 Motion Timing Scale

```typescript
export const motion = {
  micro: 0.18,
  fast: 0.32,
  normal: 0.55,
  slow: 0.85,
  cinematic: 1.35,
  easeOut: [0.22, 1, 0.36, 1],
  easeInOut: [0.65, 0, 0.35, 1],
}
```

## 9.3 Mobile Motion Rules

- Disable magnetic effects on touch devices.
- Reduce scroll scrub complexity.
- Replace heavy pinned sections with stacked cards under 768px.
- Keep CTA visible and tappable.
- Use `100dvh`, not only `100vh`.

## 9.4 Reduced Motion

Add CSS:

```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    scroll-behavior: auto !important;
    transition-duration: 0.01ms !important;
  }
}
```

In JS:

```typescript
const prefersReducedMotion = window.matchMedia('(prefers-reduced-motion: reduce)').matches
if (prefersReducedMotion) {
  // skip GSAP timelines, show final states
}
```

export function getDeviceTier() {
  if (typeof navigator === 'undefined') return 'high'
  const memory = (navigator as any).deviceMemory || 4
  const cores = navigator.hardwareConcurrency || 4
  const isTouch = matchMedia('(pointer: coarse)').matches

  if (memory <= 2 || cores <= 4) return 'low'
  if (isTouch && memory <= 4) return 'mid'
  return 'high'
}
